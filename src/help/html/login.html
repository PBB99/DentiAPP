1 ¿Cómo acceder a DentiApp?

Para acceder a esta aplicación rellena el formulario, con las creedenciales correspondientes, que se muestra a continuación:



Vea cada uno de los componenetes del formulario, empezando desde arriba:

Usuario: Se trata del nombre de usuario proporcionado para acceder a la aplicación.
Password: Contraseña asociada al usuario para acceder a la aplicación
Iniciar sesión: Botón para acceder a la aplicación.
