Módulo Usuarios


En este módulo se podrá gestionar las operaciones referentes a usuarios:

1. Visualizar usuarios
2. Añadir un usuario
3. Modificar un usuario


1 Visualizar usuarios

Los usuarios se pueden visualizar en la tabla principal que aparece en el centro de la pantalla. Haciendo click izquierdo en la parte superior izquierda en “Admin” o “Doctor” se puede cambiar entre los usuarios administradores y los usuarios doctores. La tabla que este cargada se mostrará con la letra en negro. Se puede usar el buscador que se encuentra al lado de la lupa en la parte superior izquierda para buscar usuarios por su nombre, dni o apellido.



2 Añadir usuario

Para añadir usuarios se pulsará el botón representado por una cruz verde que se encuentra en la esquina superior derecha de la tabla de usuarios.



A continuación, se abrirá una ventana para completar la siguiente información del usuario.

Dni del usuario que servirá como identificador de usuario para iniciar sesión.
Nombre del usuario.
Apellido del usuario.
Especialidad que se le va asignar al usuario.
Contraseña del usuario para iniciar sesión.



Una vez que los campos hayan sido relleneados pulsando el botón “OK” de la parte inferior se añadirá el usuario y se actualizará la tabla de ususarios con la nueva información.

También se puede cancelar en cualquier momento la operación de añadir usuario pulsando el botón de “Cancel” de la parte inferior.


3 Modificar usuario

Para modificar un usuario es necesario seleccionar haciendo click izquierdo en un usuario ya existente, posteriormente se pulsará el botón representado por un lápiz negro que se encuentra en la esquina superior derecha de la tabla de usuarios.



A continuación, se abrirá una ventana para completar la siguiente información del usuario.

Dni del usuario que servirá como identificador de usuario para iniciar sesión.
Nombre del usuario.
Apellido del usuario.
Especialidad que tenia el usuario.
Nueva Especialidad que se le va asignar al usuario.
Contraseña del usuario para iniciar sesión.
Estado que servira para darlo de alta o de baja.



Una vez que los campos hayan sido modificados pulsando el botón “OK” de la parte inferior se modificara el usuario y se actualizará la tabla de usuarios con la nueva información.

También se puede cancelar en cualquier momento la operación de modificar usuario pulsando el botón de “Cancel” de la parte inferior.
